# Personal Expense Tracker (Java Swing)

## Description
A desktop application to track personal income and expenses.  
Built with **Java Swing**, it allows users to add transactions, view them in a table, and see a summary.

## Features
- Add Income / Expense
- View transactions in a JTable
- View Total Income, Expense, and Balance
- Colored bars to visualize Income vs Expense

## How to Run
1. Open terminal in `src` folder
2. Compile all Java files:
  ```
javac *.java
```  
3.Run the Project : 
```
java Main
```
